Instructions to run

1) The Program is written in JAVA using Eclipse IDE
2) To execute the same open Eclipse IDE and then click on File-->Import-->General-->Exisisting Projects into Workspace -->Browse and select the BinarySearchTree Directory
3) The main function is in Main.java
4) Click on Run and the Program will execute

Else 

The same can be executed using the command prompt or Terminal

1) Go to the path
2) To compile command will be javac Main.java 
3) To run java Main.java
